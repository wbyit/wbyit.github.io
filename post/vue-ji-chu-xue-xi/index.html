Vue是什么？

Vue是用来构建界面的一套渐进式框架
与其他重量级框架不同的是，Vue的核心库只关心视图层
Vue完全有能力驱动采用单文件组件和Vue生态系统支持的库开发的复杂单页应用。

Vue的目标是通过尽可能简单的 API 实现响应的数据绑定 和 组合的视图组件。
